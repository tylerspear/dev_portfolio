This is an app I developed as a longtime fan of the game show Jeopardy. I always enjoy finding trivia games to practice my general knowledge skills, but am often turned off by the freemium features. When I found the jservice.io API which contains a catalog of historical Jeopardy questions, I knew this was something I wanted to use in a web app build. There are 2 basic features of my app - a random selection from the question catalog, or an option to find questions based on certain categories.
